# @unrouted/core

The core engine of [Unrouted](https://github.com/harlan-zw/unrouted) without any presets.

## Usage

```ts
import { createUnrouted } from '@unrouted/core'

const unrouted = await createUnrouted({ /* user options */ }, { /* provider options */ })

```
